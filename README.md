# APRS Live Decoder Tool

A Python-based tool for decoding and displaying APRS signals in real-time with an interactive map.

![APRS Decoder](https://img.shields.io/badge/APRS-Decoder-blue)
![Python](https://img.shields.io/badge/Python-3.8%2B-green)

## Overview

This tool allows:
- Live decoding of APRS signals via audio input
- Visualization of stations on an interactive OpenStreetMap
- Support for APRS symbols and station information
- Real-time log display of all received packets

## Quick Start

- Audio input device (sound card with Line-In or microphone)
